User Types and Credentials

Admin: admin / 123456
User: user / 123456

App Flow

  1. Splash Screen
  2. Terms and Conditions Acceptance
  3. Login Page

Admin Panel

  • If an admin is logged in, they will be navigated to the Admin Panel.
  • Admin can perform the following actions:
    • Create Terms and Conditions records
    • Add and manage questions
    • Add and manage users

User Page

  • If a user is logged in, they will be navigated to the User Page.
  • On the User Page, the user can:
    • Click on the play icon to listen to a question
    • A prompt will appear with "Start Speaking" to let the user start speaking
    • Once the user answers, the app will keep asking questions until the last question
    • The user must click "Done" to conclude the conversation and send the data to the Flask app API
